# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.4.0] - 2025-12-XX

### Изменено
- Проведен масштабный рефакторинг кода: разделение на модули для улучшения поддерживаемости
- Добавлена система логирования с флагом DEBUG для управления выводом в консоль (отключено для продакшена)
- Улучшена обработка ошибок: убраны лишние логи в консоль для ожидаемых ошибок (429, CORS)

## [1.3.0] - 2025-12-XX

### Добавлено
- Поддержка страниц профилей игроков (`/player/`)
- Обработка тиммейтов на страницах профилей игроков
- Улучшенное отображение ошибок с иконками и кнопками действий

## [1.2.0] - 2025-11-22

### Добавлено
- Автоматическое определение версии из package.json при создании zip-архива
- GitHub Actions workflow для автоматических релизов

### Изменено
- Версия обновлена до 1.2.0

## [1.1.2] - 2025-11-XX

### Исправлено
- Исправление бага с динамической загрузкой контента на странице матча
- Улучшена обработка ошибок при поиске профилей

### Добавлено
- Система кэширования данных игроков
- Механизм автоматической очистки устаревшего кэша (TTL 1 час)
- Popup интерфейс с функционалом очистки кэша
- Retry механизм для API запросов
- Поддержка различных форматов Steam ID

## [1.0.0] - 2025-XX-XX

### Добавлено
- Первоначальный релиз
- Автоматический поиск Faceit профилей для игроков VSCL.ru
- Отображение ELO рейтинга на страницах матчей
- Ссылки на Faceit Analyser для детальной статистики
