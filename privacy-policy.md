# VSCL Faceit Finder Privacy Policy

**Last Updated:** December 2025

## Overview

VSCL Faceit Finder is a browser extension that displays public Faceit profile information (ELO rating, nickname) for players on the VSCL.ru platform. This extension does not collect, store, or transmit any personal user data.

## Data Collection and Usage

**Important:** This extension does NOT collect any personal user data. It only retrieves and displays publicly available information.

1. **What Data We Access:**
   - Public Faceit profile data (ELO rating, nickname) - obtained from Faceit's public API
   - Public Steam IDs from VSCL.ru player profiles - obtained from publicly visible pages
   - This data is cached locally in the user's browser for performance optimization

2. **What We Do NOT Collect:**
   - Personal information (name, email, address, etc.)
   - Authentication credentials
   - Browsing history
   - User location
   - Financial information
   - Any other personal data

3. **How We Use Data:**
   - To display ELO ratings and Faceit nicknames on VSCL.ru pages
   - For local caching to optimize performance and reduce API requests
   - All data is used exclusively for the extension's core functionality

4. **Data Storage:**
   - All cached data is stored locally in the user's browser using Chrome's storage API
   - Cache can be cleared by the user at any time through the extension's popup interface
   - We do NOT store data on external servers
   - All cached data is automatically deleted when the extension is removed

## Data Sharing and Transmission

1. **Data Sharing:**
   - We do NOT share, sell, or transmit any data to third parties
   - Data never leaves the user's browser except for API requests to fetch public information
   - We do NOT use analytics, trackers, or advertising services
   - We do NOT collect or transmit data for any purpose other than displaying public profile information

2. **API Requests:**
   - Faceit API requests are made only to fetch public profile data (ELO, nickname)
   - VSCL.ru requests are made only to obtain publicly visible Steam IDs from player profiles
   - All API requests are executed locally in the user's browser
   - No data is sent to our servers or any third-party servers

## Security and User Rights

1. **Data Protection:**
   - All cached data is stored locally in the user's browser
   - We do NOT use remote code or execute code from external sources
   - The extension does NOT have access to user's personal data, passwords, or authentication credentials
   - The extension only accesses publicly available information from VSCL.ru and Faceit API

2. **User Rights:**
   - Users can clear the cache at any time through the extension's popup interface
   - Users can disable or remove the extension at any time
   - All cached data is automatically deleted when the extension is removed
   - Users have full control over their data

3. **Permissions:**
   - **Storage permission:** Used only for local caching of public profile data
   - **Host permissions:** Used only to access VSCL.ru pages and Faceit API for fetching public data
   - No other permissions are requested or used

## Policy Changes

We reserve the right to update this privacy policy. When significant changes are made, we will notify users through extension updates.
